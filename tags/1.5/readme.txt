پلاگین را نصب کنید

سپس داخل فیلد پلاگین لینک دانلود مستقیم پلاگین مورد نظر خود را وارد کنید.

بعد روی دکمه نصب کلیک کنید تا پلاگین مورد نظر دانلود و نصب شود.

توجه کنید لینک باید حاوی فایل اصلی پلاگین باشد ( یعنی فقط فایل های پلاگین داخلش باشه نه اینکه پلاگین داخل یک فایل دیگه و راهنما ها باشه )

لینک های غیر مستقیم توسط پلاگین دانلود نمیشود.

اگر خوشتون اومد و دوست داشتید نظر بدید و امتیاز 5 ستاره ثبت کنید.

== Frequently Asked Questions ==
= آیا برای آپدیت پلاگین ها هم میشه استفاده کرد =
بله شما میتوانید داخل فیلد لینک دانلود مستقیم پلاگین جدید رو بگذارید تا پلاگین رو براتون دانلود و آپلود کنه
= استفاده از این پلاگین روی سرعت سایت تاثیر داه ؟ =
خیر این پلاگین در بک اند یا همان صفحه مدیریت فراخوانی میشه و در حال حاضر کمتر از 50 کیلوبایت حجم داره
