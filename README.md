# EZ-Downloader

**EZ-Downloader** یک افزونه سبک، ساده و کاربردی برای وردپرس است که امکان نصب و به‌روزرسانی افزونه‌ها را از طریق **لینک دانلود مستقیم فایل ZIP** فراهم می‌کند.

با استفاده از این افزونه، مدیر سایت می‌تواند بدون نیاز به دانلود دستی فایل، بدون ورود به File Manager و بدون استفاده از FTP، فقط با وارد کردن لینک مستقیم فایل ZIP، افزونه مورد نظر را روی وردپرس نصب یا آپدیت کند.

---

## معرفی افزونه

در وردپرس، برای نصب یک افزونه خارج از مخزن رسمی معمولاً باید فایل ZIP افزونه را ابتدا دانلود کنید، سپس وارد پیشخوان وردپرس شوید و آن را به صورت دستی آپلود کنید.

EZ-Downloader این روند را ساده‌تر می‌کند.

کافی است لینک مستقیم فایل ZIP افزونه را وارد کنید تا افزونه به صورت خودکار فایل را دریافت کرده و عملیات نصب را انجام دهد.

این افزونه مخصوص بخش مدیریت وردپرس است و در ظاهر سایت یا فرانت‌اند چیزی نمایش نمی‌دهد.

---

## کاربرد اصلی

EZ-Downloader برای نصب یا آپدیت افزونه‌های وردپرسی از طریق لینک مستقیم ساخته شده است.

این افزونه مخصوص زمانی است که:

- لینک مستقیم فایل ZIP افزونه را دارید.
- نمی‌خواهید فایل را دستی دانلود و آپلود کنید.
- می‌خواهید افزونه‌ای خارج از مخزن وردپرس نصب کنید.
- می‌خواهید نسخه جدید یک افزونه را دستی آپدیت کنید.
- به FTP یا File Manager دسترسی ندارید.
- مدیریت چند سایت وردپرسی را برعهده دارید و نصب دستی برایتان زمان‌بر است.

## نحوه نصب از پیشخوان وردپرس

برای نصب افزونه از مخزن وردپرس:

1. وارد پیشخوان وردپرس شوید.
2. به بخش **افزونه‌ها > افزودن** بروید.
3. عبارت **EZ-Downloader** را جستجو کنید.
4. افزونه را نصب کنید.
5. پس از نصب، افزونه را فعال کنید.

---

## نحوه نصب دستی

برای نصب دستی افزونه:

1. فایل ZIP افزونه را دانلود کنید.
2. وارد پیشخوان وردپرس شوید.
3. به مسیر **افزونه‌ها > افزودن > بارگذاری افزونه** بروید.
4. فایل ZIP را انتخاب کنید.
5. روی نصب کلیک کنید.
6. افزونه را فعال کنید.

---

## نحوه استفاده

بعد از نصب و فعال‌سازی افزونه:

1. وارد پیشخوان وردپرس شوید.
2. وارد صفحه EZ-Downloader شوید.
3. لینک مستقیم فایل ZIP افزونه مورد نظر را وارد کنید.
4. روی دکمه نصب کلیک کنید.
5. منتظر بمانید تا فایل دانلود و نصب شود.

## لینک مستقیم یعنی چه؟

لینکی که در EZ-Downloader وارد می‌کنید باید لینک مستقیم دانلود فایل ZIP باشد.

یعنی وقتی لینک را در مرورگر باز می‌کنید، فایل ZIP باید مستقیماً دانلود شود، نه اینکه وارد یک صفحه دانلود، صفحه تبلیغاتی یا صفحه تأیید شوید.

### نمونه لینک مناسب

```text
https://example.com/plugin-name.zip
```

### نمونه لینک نامناسب

```text
https://example.com/download-page
```

یا:

```text
https://example.com/file?id=123
```

اگر لینک به جای فایل ZIP، یک صفحه HTML یا صفحه دانلود غیرمستقیم برگرداند، افزونه نمی‌تواند آن را به درستی نصب کند.

## ساختار صحیح فایل ZIP

فایل ZIP باید شامل فایل‌های اصلی افزونه وردپرس باشد.

ساختار مناسب:

```text
plugin-folder/
├── plugin-name.php
├── includes/
├── assets/
└── readme.txt
```

ساختار نامناسب:

```text
download-package.zip
└── another-folder/
    └── plugin-folder.zip
```

یا:

```text
plugin-folder.zip
└── plugin-folder/
    └── another-plugin-folder/
        └── plugin-name.php
```

فایل ZIP باید به شکلی باشد که وردپرس بتواند آن را به عنوان افزونه شناسایی و نصب کند.

## آپدیت دستی افزونه‌ها

EZ-Downloader فقط برای نصب افزونه جدید نیست.

اگر نسخه جدید یک افزونه را به صورت فایل ZIP در اختیار دارید، می‌توانید لینک مستقیم آن را در افزونه وارد کنید تا نسخه جدید دریافت و نصب شود.

این قابلیت برای افزونه‌هایی مفید است که:

- در مخزن رسمی وردپرس نیستند.
- آپدیت خودکار ندارند.
- از سایت توسعه‌دهنده دانلود می‌شوند.
- نسخه جدید آن‌ها فقط با لینک مستقیم ارائه می‌شود.
- روی چند سایت باید به صورت دستی نصب یا آپدیت شوند.

## محدودیت‌ها

EZ-Downloader فقط با لینک مستقیم فایل ZIP کار می‌کند.

موارد زیر ممکن است توسط افزونه نصب نشوند:

- لینک‌هایی که مستقیم به فایل ZIP ختم نمی‌شوند.
- لینک‌هایی که نیاز به ورود کاربر دارند.
- لینک‌هایی که نیاز به کپچا دارند.
- لینک‌هایی که ابتدا وارد صفحه دانلود می‌شوند.
- لینک‌هایی که فایل واقعی را بعد از چند مرحله ارائه می‌کنند.
- فایل‌هایی که ساختار استاندارد افزونه وردپرس ندارند.
- فایل‌هایی که افزونه اصلی داخل یک ZIP دیگر قرار گرفته است.

## نکات امنیتی

قبل از نصب هر افزونه از طریق لینک مستقیم، مطمئن شوید فایل از منبع معتبر دریافت می‌شود.

برای امنیت بیشتر:

- فقط از منابع قابل اعتماد استفاده کنید.
- قبل از نصب افزونه‌های ناشناس، از سایت بکاپ بگیرید.
- فایل‌هایی که از منابع نامعتبر دریافت می‌شوند را نصب نکنید.
- افزونه‌های نال‌شده یا مشکوک را نصب نکنید.
- پس از نصب افزونه، عملکرد سایت را بررسی کنید.

EZ-Downloader فقط فرآیند نصب از لینک مستقیم را ساده‌تر می‌کند. مسئولیت بررسی امنیت و اعتبار افزونه‌ای که نصب می‌شود با مدیر سایت است.

## سوالات متداول

### آیا EZ-Downloader برای نصب قالب هم استفاده می‌شود؟

خیر. این افزونه برای نصب افزونه‌های وردپرسی طراحی شده است.

### آیا لینک باید مستقیم باشد؟

بله. لینک باید مستقیماً فایل ZIP افزونه را دانلود کند.

### آیا می‌توان با این افزونه افزونه‌ها را آپدیت کرد؟

بله. اگر لینک مستقیم نسخه جدید افزونه را داشته باشید، می‌توانید برای آپدیت دستی از آن استفاده کنید.

### آیا افزونه در فرانت‌اند سایت چیزی نمایش می‌دهد؟

خیر. افزونه فقط در بخش مدیریت وردپرس استفاده می‌شود.

### آیا برای استفاده از افزونه نیاز به تنظیمات خاصی است؟

خیر. افزونه ساده طراحی شده و نیاز به تنظیمات پیچیده ندارد.

### چرا بعضی لینک‌ها نصب نمی‌شوند؟

معمولاً به این دلیل که لینک مستقیم نیست، فایل ZIP معتبر نیست، یا ساختار فایل ZIP برای وردپرس استاندارد نیست.
